# Hospital Inpatient Discharge Dashboard
## Streamlit Data Analytics Project | Healthcare Performance Insights

### Project Overview

This project is build to present an interactive dashboard using Python, Streamlit, Pandas, and Plotly to analyze hospital inpatient discharge data.
The goal of this project was to help hospital administrators, clinicians, and operations managers understand patient discharge patterns and improve decision-making through data insights.

### Features

Upload dataset (CSV / Excel)

Automatic data cleaning

KPI metrics (LOS, Charges, Discharges, Readmission rate)

Filters for facility, diagnosis, severity, county & payment type

Interactive charts (Bar, Box, Heatmap, Pie, Histogram)

Diagnosis leaderboard and data download option

Streamlit cloud deployment

Technologies

Python • Pandas • NumPy • Plotly • Streamlit • GitHub
