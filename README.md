# Hospital Inpatient Discharge Dashboard
## Streamlit Data Analytics Project | Healthcare Performance Insights
ðŸ“Œ Project Overview

This project presents an interactive dashboard built using Python, Streamlit, Pandas, and Plotly to analyze hospital inpatient discharge data.
The goal is to help hospital administrators, clinicians, and operations managers understand patient discharge patterns and improve decision-making through data insights.

The dashboard allows users to explore:

Length of stay trends
Patient charges and severity comparison
Readmission rates
Diagnosis-based performance
Facility and county-level differences

Key Features

Upload hospital discharge dataset (CSV or Excel)
Automatic data cleaning and processing
KPI summary (Total discharges, Average LOS, Charges, Readmission rate)
Filters by facility, county, diagnosis, severity, payment type, and date range
Interactive visualizations including:
Bar chart: Average stay per diagnosis
Box plot: Charges by severity
Heatmap: Length of stay by facility & county
Pie chart: Payment type distribution
Histogram: Length-of-stay distribution
Diagnosis leaderboard and export options
Download cleaned dataset and insights CSV
